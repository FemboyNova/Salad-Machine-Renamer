# Salad Machine Renamer

## Overview
Salad Machine Renamer replaces machine IDs on Salad.com's earn page with custom, human-readable names to make monitoring and identification easier.

## Features
- Replace default machine IDs with custom names
- Persist custom names across browser sessions
- Works on Salad.com's earn page
- Compatible with Chromium-based browsers and Firefox

## Installation

### Chrome
Option 1: Chrome Web Store
1. Install from the Chrome Web Store: https://chromewebstore.google.com/detail/apehjenffjjamcpghbahankhcheepkdn
2. Click the extension icon and configure your machines.

Option 2: Manual (Developer Mode)
1. Download the repository ZIP from GitHub and extract it.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable "Developer mode".
4. Click "Load unpacked" and select the extracted folder containing `manifest.json`.

### Firefox
Option 1: Firefox Add-ons
1. Install from the Firefox Add-ons site: https://addons.mozilla.org/en-GB/firefox/addon/salad-machine-renamer/
2. Click the extension icon to configure it.

Option 2: Temporary (for testing)
1. Download and extract the repository.
2. Open Firefox and go to `about:debugging#/runtime/this-firefox`.
3. Click "Load Temporary Add-on..." and select `manifest.json`.

## Usage
1. Open Salad.com's earn page.
2. Click the extension icon.
3. Add a Machine ID and assign a custom name.
4. Save changes; machine names will update on the page.

## Troubleshooting
- If names do not appear, refresh the earn page.
- Confirm the Machine ID is entered correctly.
- Check browser console for errors and report issues with details.
